# Requirements

SAP_BASIS 7.51 because of ABAP Test Double Frameworks (ATDF) 

(Interface if_osql_test_environment)

# Namespace

Z%TDC4ABAP%

- IF = Interface
- CL = Class
- CX = Exception Class
- TH = Test Helper Class

Example:
- ZIF_TDC4ABAP_ORDER_DAO
- ZCL_TDC4ABAP_ORDER_DAO
- ZCX_TDC4ABAP
- ZTH_TDC4ABAP_ITEM
